# 🚀 InterviewIQ.AI

**InterviewIQ.AI** is a full-stack SaaS-style AI interview preparation platform that helps users prepare for technical and HR interviews using AI-generated questions and personalized feedback.

The platform allows users to upload their resume, receive tailored interview questions, practice interviews, and get intelligent feedback. It also includes a credit-based system with online payments, making it a production-ready SaaS application.

---

## ✨ Features

* 📄 Upload Resume (PDF)
* 🤖 AI-generated interview questions based on your resume
* 💼 Practice Technical & HR interview rounds
* 📝 Receive AI-powered interview feedback
* 💳 Credit-based usage system
* 💰 Purchase additional credits through Razorpay
* 🔐 Secure Google Authentication using Firebase
* 🎨 Modern, responsive UI with smooth animations using Framer Motion
* ☁️ Full-stack deployment ready for production using Render

## 📂 Project Overview

InterviewIQ.AI follows a modern SaaS architecture consisting of:

* Frontend built with React.js
* REST API powered by Express.js
* MongoDB for data storage
* Firebase Authentication for secure login
* Razorpay payment integration
* AI-powered interview generation and feedback
* Credit management system for controlled AI usage
